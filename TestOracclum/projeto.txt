Para avaliação técnica você terá uma semana para realizar este projeto e nos enviar via git.

- Crie uma página em HTML representando uma propaganda do seu filme/série favorito/a. Use sua criatividade para criar uma página chamativa, não utilize apenas textos.
- feed com noticias(twitter)


- Nessa página deve existir uma funcionalidade para que quem estiver navegando consiga enviar perguntas sobre o filme/série para o criador dá página
- Crie um backend para receber, buscar e responder as perguntas
- Crie um frontend para realizar o login, visualizar uma página com as perguntas e responde-las

### Tech

- Qualquer Database de sua escolha
- Página Propaganda: HTML, CSS e JS
- Frontend App: React
- Backend App: Node.js

### Backendcd

- Criar uma API com os seguintes endpoints:
    - login
    - criar-pergunta
    - buscar-perguntas
    - responder-pergunta

    regras de negocio: login e perguntas
    
- Não é necessário criar um endpoint para criar conta, o usuário pode ser criado direto na database
